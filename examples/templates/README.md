# Industry Templates

This folder contains starter ACP bundles for common adoption paths.

## Bundles
- `hiring/`: candidate review and decision handoff
- `outsourced-production/`: client/vendor revision flow with delegation and settlement intent
- `research/`: evidence-backed investigation and verification
- `marketing/`: campaign approval, activation, and payout intent
- `internal-multi-agent/`: internal agent delegation with reservation and freeze support

## Notes
- These are starter templates, not exhaustive production playbooks.
- Files are intentionally small so they can be copied into a new workspace and adapted quickly.
- Every YAML file here should remain valid against the current `schemas/core` snapshot.
