# Examples

This directory contains reference artifact bundles.

## Bundles
- `minimal-task/`: smallest accountability flow
- `delegated-research/`: parent-child delegation with reservation and dispute freeze
- `integrations/`: reference ACP harness integration skeleton (FastAPI + subprocess adapter)
- `templates/`: industry-specific starter bundles for direct adoption

## Notes
- These examples are illustrative and intentionally small.
- They align with current `schemas/core` and `schemas/meta` snapshots.
