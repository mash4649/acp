# ACP 30-Second Demo Script

## Script
"ACP is an accountability layer for delegated AI work. It does not replace execution frameworks, transport protocols, or payment rails. In ACP, prompt is not contract, claim is not evidence, and verification is not settlement. We model delegated work with append-only events, evidence-linked verification, explicit settlement intent, and dispute/freeze hooks so accountability stays interoperable across existing stacks."
