# ACP 30秒デモ台本（日本語）

## 台本
「ACP は委任された AI 作業のアカウンタビリティ層です。実行フレームワーク、トランスポートプロトコル、決済レールを置き換えるものではありません。ACP では プロンプト は 契約 ではなく、主張 は 証拠 ではなく、検証 は 精算 ではありません。追記専用 なイベント、証拠連動検証、明示的な 精算意図、そして 紛争/凍結 フックによって、既存スタック横断のアカウンタビリティを保ちます。」

## 表示順
1. `agreement_v1` と `revision_v1`
2. `event_v1` + `evidence_pack_v1`
3. `verification_report_v1` の後に `settlement_intent_v1` を別表示
4. `freeze_record_v1` を dispute 分岐として表示

## 利用場面
- ポッドキャスト/インタビュー冒頭
- SNS ティーザー動画
- スライド1枚目のナレーション
