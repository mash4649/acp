# 比較マトリクス（短縮版・日本語）

|---|---|---|---|---|---|
| 主目的 | 委任作業のアカウンタビリティ | ツール/文脈アクセス | エージェント通信 | 実行オーケストレーション | 価値移転/ファイナリティ |

## 要点
ACP はアカウンタビリティ意味論を標準化してスタックを補完するもので、トランスポート/実行/ペイメント を置き換えるものではありません。
