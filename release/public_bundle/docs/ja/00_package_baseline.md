# ACP 公開編集ベースライン（日本語）

## 公開定義
- ACP は実行フレームワークではありません。
- ACP はトランスポートプロトコルではありません。
- ACP は決済レールではありません。
- ACP は委任された AI 作業のアカウンタビリティ層です。
- プロンプトは契約ではありません。
- 主張は証拠ではありません。
- 検証は精算ではありません。
- 追記専用の因果イベント履歴が必要です。
- ポリシー評価は再実行可能である必要があります。
- 狭いコア、強い不変条件、拡張可能なコンパニオン/プロファイルを採用します。

## コアアーティファクト（15 固定）
- agreement_v1
- revision_v1
- event_v1
- evidence_pack_v1
- proof_artifact_v1
- verifier_descriptor_v1
- proof_finality_v1
- resource_reservation_v1
- delegation_edge_v1
- data_policy_binding_v1
- effective_policy_projection_v1
- time_fact_v1
- verification_report_v1
- settlement_intent_v1
- freeze_record_v1

## 非目標ベースライン
- ACP を MCP/A2A/ワークフロー/ペイメントの代替として売り込みません。
- 証拠/検証/精算/紛争を統合しません。
- Core を 7 アーティファクト等へ再定義しません。

## パッケージ表記ルール
公開文書では、必要に応じて本ファイルと同等の front-matter 情報（状態・種別・出典）を維持してください。
